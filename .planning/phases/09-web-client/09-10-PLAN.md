## Problem Summary

The static export build (`pnpm nx build web`) fails with:
```
TypeError: Cannot read properties of null (reading 'useEffect')
```

**Root Cause:** Providers.tsx is a client component with `useEffect` for service worker registration. Static export mode prerenders all pages at build time, but React hooks cannot execute during SSR/prerendering.

**User Decision:** Remove PWA features (Option 3 from BLOCKER.md) - Keep static export, sacrifice offline support and installability.

## Current Files to Modify

1. **Providers.tsx** - Remove service worker registration useEffect
2. **sw.js** - Delete (no longer needed)
3. **OfflineBanner.tsx** - Delete (relies on SW for offline detection)
4. **browser.ts** - Delete (only used for SW browser check)

## Files to Keep (Harmless)

- **manifest.json** - May be used by browsers for basic PWA metadata
- **icon-*.png** - Icons can still be used for bookmarks/tabs
</context>
