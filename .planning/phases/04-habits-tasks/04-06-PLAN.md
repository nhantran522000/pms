## Filtering Options

| Status | Condition |
|--------|-----------|
| pending | isCompleted = false AND (dueDate IS NULL OR dueDate >= today) |
| completed | isCompleted = true |
| overdue | isCompleted = false AND dueDate < today |

## Sorting Options

| Sort By | Default Order | Fields |
|---------|---------------|--------|
| dueDate | asc | dueDate (nulls last) |
| priority | desc | priority |
| createdAt | desc | createdAt |
| title | asc | title |
