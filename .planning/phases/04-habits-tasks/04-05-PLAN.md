## AI Gateway Integration

From Phase 2, the AI Gateway supports EXTRACT task type:
- Input: prompt with text to extract entities from
- Output: ExtractResult with entities array
- Use AiGatewayService.execute() with taskType: 'EXTRACT'

## Natural Language Patterns to Support

| Input | Extracted |
|-------|-----------|
| "Buy milk tomorrow 5pm #errands" | title: "Buy milk", dueDate: tomorrow 5pm, tags: ["errands"] |
| "Call mom next monday #family" | title: "Call mom", dueDate: next Monday, tags: ["family"] |
| "Submit report in 3 days #work urgent" | title: "Submit report", dueDate: in 3 days, tags: ["work"], priority: 4 |
| "Review PR friday #work" | title: "Review PR", dueDate: this Friday, tags: ["work"] |
| "Pay rent on the 1st #bills" | title: "Pay rent", dueDate: 1st of next month, tags: ["bills"] |

## Date Parsing Logic

1. First try AI extraction for complex expressions
2. Fallback to regex for common patterns:
   - "tomorrow" -> tomorrow's date
   - "next monday" -> next occurrence of Monday
   - "in N days/hours" -> relative date
   - "on the Nth" -> Nth day of current/next month
